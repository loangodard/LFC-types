Pour importer le module dans les autres projets LFC :
`npm install git+https://github.com/loangodard/LFC-types.git`

Ne pas oublier d'exécuter `npm run build` avant de push les types sur git.

Pour forcer la réinstalation les types sur un micro service : 
`npm install git+https://github.com/loangodard/LFC-types.git --force`
